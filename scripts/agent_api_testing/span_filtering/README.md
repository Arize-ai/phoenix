# Span Filtering Trial

Measures whether independent LLM agents can construct correct
`GET /v1/projects/{id}/spans?attribute=...` URLs from just the OpenAPI
schema for realistic OpenInference-shaped queries.

Work item: [`_work/agent-usability-testing-attribute-filter-param/`](../../../../../.lore/repos/github.com/arize-ai/phoenix/_work/agent-usability-testing-attribute-filter-param).

## Prerequisites

- A local Phoenix reachable at `http://localhost:6006` (or pass `--base-url`).
- `claude` and/or `codex` CLIs on `$PATH`, authenticated.

## Run

```sh
# 1. Seed fixtures (idempotent — deletes and recreates the trial project).
uv run scripts/agent_api_testing/span_filtering/seed.py

# 2. Run the full trial matrix (4 agents × 7 prompts by default).
uv run scripts/agent_api_testing/span_filtering/run_trial.py

# Or scope it:
uv run scripts/agent_api_testing/span_filtering/run_trial.py \
  --agents claude:claude-sonnet-4-6,codex \
  --prompts P1,P2,P5
```

Run `run_trial.py --help` for all flags. Each invocation runs in a fresh
temp cwd with no persisted session — `claude -p --no-session-persistence
--setting-sources ""` and `codex exec --ephemeral --skip-git-repo-check`.

## Outputs

| Path | Content |
|---|---|
| `manifest.json` | Seeded spans + ground-truth expected match set per prompt. |
| `_work/agent-usability-testing-attribute-filter-param/trial-runs/<agent>/<prompt>.jsonl` | Raw event stream per trial. |
| `_work/agent-usability-testing-attribute-filter-param/trial-runs/summary.json` | Aggregated URL attempts, final answers, wall times. |
| `_work/agent-usability-testing-attribute-filter-param/trial-results.md` | Scored per-prompt report with remediation recommendation. |

## Files

- `seed.py` — populates the `agent-trial-attribute-filter` project with
  6 OpenInference-shaped spans.
- `prompts.md` — P1–P7 (plus optional P8) with classification rubric. The
  prompt text is what the agent sees; the rubric below each prompt is for
  scoring only — **never paste into the agent**.
- `run_trial.py` — the harness; invokes agents, parses their tool-use
  streams, records URL attempts + final answers.
- `manifest.json` — regenerated by `seed.py` each run.

## Notes

Trials are non-deterministic and cost real API dollars. Not in CI. Rerun
`seed.py` between trials to reset state.
