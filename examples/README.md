# Phoenix Examples

A collection of examples demonstrating how to use Phoenix.

- [support-bot-evals](./support-bot-evals) — a realistic eval suite for a
  customer-support FAQ bot, written in both Python (pytest) and TypeScript
  (Vitest): real LLM calls, an LLM-as-judge metric, and CI gating on
  suite-level acceptance criteria.

Have an interesting use-case for phoenix? Submit a PR to add it to the list!
