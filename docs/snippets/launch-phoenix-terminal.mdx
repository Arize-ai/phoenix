**Launch your local Phoenix instance:**

```bash
uvx arize-phoenix serve
```

If you don't have [uv](https://docs.astral.sh/uv/) installed, use `pip install arize-phoenix && phoenix serve` instead.

This serves the web UI and the OTLP HTTP endpoint on port 6006, and the OTLP gRPC endpoint on port 4317.

For details on customizing a local terminal deployment, see [Terminal Setup](/docs/phoenix/environments#terminal).
