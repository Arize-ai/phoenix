**Available in arize-phoenix 15.5.0+**

The Phoenix Playground now supports **Google Cloud Vertex AI** as a first-class provider — Gemini and Claude models, authenticated via Application Default Credentials.

## What's new

- **Native Vertex AI provider** in the playground, settings, prompt management, and dataset LLM-evaluator surfaces
- **Application Default Credentials**: no API key required. Works locally with `gcloud auth application-default login` and on GCP workloads with attached service accounts.
- **Gemini and Claude**: a single Vertex AI provider covers both Gemini (via `google-genai`) and Claude (via Anthropic's Vertex SDK)
- **Distinct OpenInference attribution**: Vertex traffic emits spans with `llm.provider="vertex_ai"`, separate from AI Studio Gemini (`google`)
