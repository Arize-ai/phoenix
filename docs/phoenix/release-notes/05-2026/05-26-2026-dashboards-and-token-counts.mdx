# Dedicated Dashboards Page

May 26, 2026

**Available in arize-phoenix 16.1.0+**

Project metric charts are now available from a top-level **Dashboards** navigation item. Previously, dashboards were accessible only from within a project. The new route lets you jump straight to any project's metrics from anywhere in Phoenix, with a searchable project menu and a shared time-range selector.

- **Persistent selection**: Phoenix remembers the last project you viewed and returns to it on the next visit.
- **Shareable links**: `/dashboards/projects/:projectId` links resolve to the correct project, including projects outside the first paginated page.
- **Searchable project menu**: filter projects by name directly from the selector.
