title: "04.14.2026 Phoenix AI Assistant Improvements"
description: "Customize the system prompt, toggle the assistant on and off, and rate responses with thumbs up/down feedback."
---

Several improvements to the Phoenix AI assistant ship in arize-phoenix 14.3.0 through 14.5.0.

## Settings Page

**Available in arize-phoenix 14.3.0+**

The Phoenix AI assistant now has a dedicated settings page at **Settings → Agents**. Use it to customize the system prompt that shapes how the assistant responds to your queries.

## Toggle the Assistant On or Off

**Available in arize-phoenix 14.5.0+**

The assistant can be disabled from the settings page. When disabled, the assistant widget is hidden and no chat sessions are started.

## Feedback on Responses

**Available in arize-phoenix 14.5.0+**

Each assistant response now shows an action bar with thumbs up/down feedback, a copy button, and a link that opens the underlying trace in a new tab. Feedback is written as a `user_feedback` annotation on both the root span and the trace. When authenticated, annotations are keyed by username so re-submitting updates the existing entry instead of adding a duplicate.
