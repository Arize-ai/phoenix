When you open a project with no traces yet, Phoenix now shows an interactive onboarding page to help you start sending traces immediately. Select your framework, pick Python or TypeScript, and get copy-pasteable install and integration code tailored to that stack — no documentation hunting required.

## Integration-Aware Setup Guide

- **Framework selector** — choose from OpenAI, LangChain, and other supported integrations; code snippets update instantly
- **Language tabs** — switch between Python and TypeScript examples within each integration
- **Step-by-step instructions** covering package installation, environment configuration, and tracing setup
- **Inline API key generation** — create a Phoenix API key without leaving the onboarding flow
- **Live "Awaiting traces..." indicator** — the page shows a progress spinner once Phoenix detects an active connection and is waiting for the first trace to arrive

**Available in arize-phoenix 13.14.0+**
