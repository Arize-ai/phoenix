# Project Creation with Color and Description

March 26, 2026

**Available in arize-phoenix 13.19.0+**

The new project creation flow lets you assign a color and a description to a project at creation time, making it easier to distinguish projects at a glance in the project list.

- **Color picker** assigns a visual accent to the project for quick identification
- **Description field** captures the purpose of the project alongside its name
