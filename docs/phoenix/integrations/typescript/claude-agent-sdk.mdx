The [Claude Agent SDK](https://platform.claude.com/docs/en/agent-sdk/overview) gives you the same tools, agent loop, and context management that power Claude Code, programmable in Python and TypeScript. Agents built with the SDK can autonomously read files, run commands, search the web, edit code, and more.
