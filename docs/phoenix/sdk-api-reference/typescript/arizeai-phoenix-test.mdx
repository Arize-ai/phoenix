Run dataset-backed Phoenix evaluations as Vitest or Jest tests. Each
`describe()` block becomes a Phoenix dataset and a new experiment;
each `test()` becomes a dataset example and a recorded experiment run;
the assertion outcome is captured as a `pass` boolean annotation.

## Installation

```bash
npm install -D @arizeai/phoenix-test
```

<Note>
`@arizeai/phoenix-test` ships local docs and source code inside `node_modules/@arizeai/phoenix-test/`.

Browse the website version under [@arizeai/phoenix-test Docs](/docs/phoenix/sdk-api-reference/typescript/packages/phoenix-test/overview).
</Note>
