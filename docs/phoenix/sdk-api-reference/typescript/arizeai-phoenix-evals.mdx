TypeScript evaluation library for LLM applications. This package is vendor agnostic and can be used independently of any framework or platform.

<Note>
**Note**: This package is in alpha and subject to change.
</Note>

## Installation

```bash
npm install @arizeai/phoenix-evals
```

<Note>
`@arizeai/phoenix-evals` ships local docs, mirrored examples, and source code inside `node_modules/@arizeai/phoenix-evals/`.

- `docs/overview.mdx`
- `docs/examples/`
- `src/`

Browse the website version under [@arizeai/phoenix-evals Docs](/docs/phoenix/sdk-api-reference/typescript/packages/phoenix-evals/overview).
</Note>

## Local Install Pattern

After install, a coding agent can inspect:

```text
node_modules/@arizeai/phoenix-evals/docs/
node_modules/@arizeai/phoenix-evals/docs/examples/
node_modules/@arizeai/phoenix-evals/src/
```

## Recommended Starting Files

- `docs/examples/create_evaluator_example.ts` for deterministic code evaluators
- `docs/examples/faithfulness_example.ts` for a built-in LLM evaluator
- `docs/examples/classifier_example.ts` and `docs/examples/classification_example.ts` for classification flows
- `docs/examples/format_template_example.ts` for templating
- `docs/examples/phoenix_integration_example.ts` for `@arizeai/phoenix-client` integration
