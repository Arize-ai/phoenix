`@arizeai/phoenix-evals` ships ready-to-use LLM evaluator factories for common evaluation tasks.

## Start With These Installed Files

- `docs/examples/faithfulness_example.ts` for the built-in faithfulness metric
- `docs/examples/bind_evaluator_example.ts` for binding a built-in evaluator to your own record shape
- `src/llm/index.ts` for the current exported metric list

## Current Built-Ins

- `createConcisenessEvaluator`
- `createCorrectnessEvaluator`
- `createDocumentRelevanceEvaluator`
- `createFaithfulnessEvaluator`
- `createRefusalEvaluator`
- `createToolSelectionEvaluator`
- `createToolInvocationEvaluator`
- `createToolResponseHandlingEvaluator`

## Legacy Alias

`createHallucinationEvaluator` is still exported, but the code marks it as deprecated in favor of `createFaithfulnessEvaluator`.

## Example

```ts
import { openai } from "@ai-sdk/openai";
import {
  createCorrectnessEvaluator,
  createRefusalEvaluator,
} from "@arizeai/phoenix-evals";

const model = openai("gpt-4o-mini");

const correctness = createCorrectnessEvaluator({ model });
const refusal = createRefusalEvaluator({ model });
```
