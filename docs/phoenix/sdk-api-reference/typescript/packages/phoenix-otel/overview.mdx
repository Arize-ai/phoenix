`@arizeai/phoenix-otel` is the Phoenix-focused OpenTelemetry package for Node.js. It wraps provider setup, span export, instrumentation registration, and a few convenience utilities for Phoenix-specific tracing workflows.

## Install

```bash
npm install @arizeai/phoenix-otel
```

## Docs And Source In `node_modules`

After install, a coding agent can inspect the installed package directly:

```text
node_modules/@arizeai/phoenix-otel/docs/
node_modules/@arizeai/phoenix-otel/src/
```

## Best Starting Pages

- `docs/installation.mdx` and `docs/configuration.mdx` for setup and environment variables
- `docs/register.mdx` for the core provider lifecycle
- `docs/instrumentation.mdx` and `docs/manual-spans.mdx` for automatic and manual tracing
- `docs/projects-and-auth.mdx` and `docs/troubleshooting.mdx` for deployment details and debugging

## Module Map

- root export: `register`, OpenTelemetry re-exports, and utility helpers
- `register.ts`: provider setup plus attach and detach helpers
- `config.ts`: environment-variable lookup
- `utils.ts`: attribute helpers
