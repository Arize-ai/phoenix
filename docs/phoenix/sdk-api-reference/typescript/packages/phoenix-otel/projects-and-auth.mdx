## Key Settings

- `projectName` controls which Phoenix project receives the spans
- `url`, `apiKey`, and `headers` control how the exporter connects to Phoenix

## Cloud Example

```ts
import { register } from "@arizeai/phoenix-otel";

register({
  projectName: "support-bot",
  url: "https://app.phoenix.arize.com",
  apiKey: process.env.PHOENIX_API_KEY,
});
```

## Local Development Example

```ts
register({
  projectName: "support-bot-dev",
  url: "http://localhost:6006",
  batch: false,
});
```
