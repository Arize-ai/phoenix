title: "Register"
description: "Register a Phoenix tracer provider"

`register()` is the package entrypoint most applications start with. It creates a `NodeTracerProvider`, configures an OTLP exporter for Phoenix, and optionally attaches the provider globally.

## Minimal Setup

```ts
import { register } from "@arizeai/phoenix-otel";

const provider = register({
  projectName: "support-bot",
});
```
