title: "Register"
description: "Register a Phoenix tracer provider"
---

`register()` is the package entrypoint most applications start with. It creates a `NodeTracerProvider`, configures an OTLP exporter for Phoenix, and optionally attaches the provider globally.

## Start With These Installed Files

- `docs/examples/register_example.ts`
- `docs/examples/global_provider_lifecycle_example.ts`
- `src/register.ts`

## Minimal Setup

```ts
import { register } from "@arizeai/phoenix-otel";

const provider = register({
  projectName: "support-bot",
});
```

## Explicit Global Attachment

If you already created a provider and want to manage the global lifecycle yourself, use `attachGlobalTracerProvider()` and `detachGlobalTracerProvider()`.

```ts
import {
  attachGlobalTracerProvider,
  detachGlobalTracerProvider,
  register,
} from "@arizeai/phoenix-otel";

const provider = register({
  projectName: "support-bot",
  global: false,
});

const registration = attachGlobalTracerProvider(provider);
registration.detach();
detachGlobalTracerProvider();
```
