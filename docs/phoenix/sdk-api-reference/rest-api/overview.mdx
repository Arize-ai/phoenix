### Example Request

_Get dataset by ID_

<Tabs>
<Tab title="cURL">
```concurnas
curl -L \
  --url '/v1/datasets'
```
</Tab>

<Tab title="JavaScript" icon="js">
```javascript
// Get dataset by ID
const response = await fetch('/v1/datasets/{id}', {
    method: 'GET',
});

const data = await response.json();
```
</Tab>

<Tab title="Python" icon="python">
```python highlight={2}
# Get dataset by ID
import requests
response = requests.get(
    "/v1/datasets/{id}",
)

data = response.json()
```
</Tab>

<Tab title="HTTP">
```http
GET /v1/datasets/{id} HTTP/1.1
Host:
Accept: */*
```
</Tab>
</Tabs>
