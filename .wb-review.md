## Findings
1. **Missing coverage for HallucinationEvaluator in shared behavior tests**

2. **ToolSelectionEvaluator doc example used wrong input key**
   - Example payload used `agent_tool_selection` while the evaluator schema requires `tool_selection`.
   - This could mislead users following docs.

3. **HallucinationEvaluator docs not updated with new usage examples parity**
   - Args docs were updated, but examples did not include the new invocation kwargs and prompt override usage shown in other evaluators.

- Added `HallucinationEvaluator` to the shared `ALL_EVALUATORS` parameterization with warning filtering for deprecation messages.

- Updated `HallucinationEvaluator` examples to include:
  - default usage,
  - kwargs usage (e.g. `temperature=0.0`),
  - custom `prompt_template` usage.
