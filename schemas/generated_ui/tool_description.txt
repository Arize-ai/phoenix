Render a generated UI in the Phoenix chat using the available components below.
Use this tool when a compact visual UI such as metrics, charts, or an analytical card would answer the user better than prose alone.
The `spec` argument must be one complete UI tree in this shape: `{ root: string, elements: Record<string, { type: string, props: object, children: string[] }> }`.
`root` is the id of the first element to render. Each key in `elements` is an element id. `children` contains element ids from the same `elements` object.
Every element `type` must come from the component list below, and every element must include `type`, `props`, and `children`.
Do not provide partial updates, JSONL patches, markdown, or prose inside `spec`; provide the full render tree in one object.
Prefer ChartCard as the root for generated analytical UI.

AVAILABLE COMPONENTS (10):

- ChartCard: { title: string, subtitle?: string } - Card container for generated analysis, metrics, and charts. [accepts children]
- Stack: { gap?: "xs" | "sm" | "md" | "lg" } - Vertical layout for grouping generated UI components. [accepts children]
- Row: { gap?: "xs" | "sm" | "md" | "lg" } - Horizontal wrapping layout for metrics or chart summaries. [accepts children]
- Title: { text: string } - Section heading text.
- Paragraph: { text: string } - Short explanatory paragraph.
- Metric: { label: string, value: string | number, change?: string } - Single KPI or aggregate value with optional change text.
- BarChart: { title?: string, data: Array<{ label: string, value: number }> } - Simple categorical bar chart with label/value points.
- LineChart: { title?: string, data: Array<{ label: string, value: number }> } - Simple trend chart with ordered label/value points.
- PieChart: { title?: string, data: Array<{ label: string, value: number }> } - Simple proportional chart with label/value slices.
- MultiSeriesChart: { title?: string, series: Array<{ label: string, values: Array<{ label: string, value: number }> }> } - Simple placeholder for comparing multiple labeled series.
